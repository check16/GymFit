Bienvenidos a GymFit

GymFit es una aplicacion de gestion de entrenamientos donde podrás gestionar los tuyos propios así como tener un seguimiento del peso, saber cuantas Kcal consumir, etc!

Para comenzar sería útil rellenar tus datos personales tales como:

  • Nombre y apellidos
  • Altura
  • Edad

GymFit proporciona las siguientes herramientas:

Entrenamientos:
Permite crear, editar y eliminar los entrenamientos personales.
Perfil:
Permite definir datos del perfil de usuario, tales como la edad, altura, etc.
Peso corporal:
Permite registrar el peso corporal, así como llevar un seguimiento del mismo.
IDR:
Permite conocer cual es la ingesta diaria recomendada. Para ello se determina cual es la cantidad de Kcal que se debe consumir en base a los requisitos de la persona y sus objetivos.
